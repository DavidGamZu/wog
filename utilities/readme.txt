Welcome to the World of Games: The Epic Journey!

Summery:
A little python games that plays 3 games;
1. Memory Game - a sequence of numbers will appear for 1 second, and you have to guess it back.
2. Guess Game - guess a number and see if you chose like the computer.
3. Currency Roulette - try and guess the value of a random amount of USD in ILS

Each game has a difficulty that the user can choose between 1 and 5.
1. Memory Game - if the difficulty level is 1, the system will show 1 number if the difficulty is 2 the system will show 2 numbers.
2. Guess Game - the system will generate a number between 0 to the difficult number.
3. Currency Roulette - the system will be flexible to the user answer according to the of difficulty level


Instruction:
1.Install python on your computer (3.8 and above)
2.To run the app run the python file from main.py 'python3 main.py'
3.Follow the game Instruction and enjoy!
